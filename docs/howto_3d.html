HOWTO use the 3D G-code generator

The 3D script by Nicholas Peshman has been incorporated into SketchUcam so that it can use the parameters on the Parameters Dialog. It will fetch bit diameter, stepover% and multipass depth from the dialog.
  1. Move Group or groups (of 3D objects) to within the safe area starting from 0,0 and on the 0 plane
  2. Tick the 'Generate 3D Gcode' box in the parameters.
  3. Click 'Generate G-code'
WARNING: This script is still under development and does not produce perfect output, yet.
