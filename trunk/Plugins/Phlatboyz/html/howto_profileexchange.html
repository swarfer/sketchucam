HOWTO exchange tool profiles with friends


Tool and material profiles can be exchanged between users, sharing the ease of setup for particular materials. This document will guide you through exchangeing profiles with other users.
NOTE: Profiles only work with Sketchup units set to millimeters or inches

Exchanging profiles
Giving your profiles away The profiles are stored in the Sketchup folder within each users APPDATA folder which you can display via the Tools|Phlatboyz|Display Profiles Folder menu item. [MAC: see bottom of page].
Should that menu item not open a file browser in the relevant folder containing a number of .TPR files follow the instructions below
The path to this folder varies between Windows versions but can easily be found by doing the following:
do this
Click 'Start'
Click 'Run'
Paste '%APPDATA%/Sketchup'
Press Enter
or this
Open 'My Computer'
Paste '%APPDATA%/Sketchup' in the address bar
Press Enter
or this
Use 'My Computer' to navigate to (may vary between Windows versions)
C:\Users\yourusername\AppData\Roaming\SketchUp
Windows will open a file explorer with the folder displayed. All the 'somename.TPR' files are your profiles. To give them to someone else, simply email them, transfer them with a flash stick or post them on phlatforum.com in the 'Material/Bit Specific Feedrates' topic.
Installing a downloaded profile To install a profile that you have downloaded:
  1. Open your APPDATA folder (as above)
  2. Copy or move the downloaded profile file to that folder.
  3. Open Sketchup and confirm that you can load it.

Mac/Linux users: you will be running Sketchup under Wine and the APPDATA environment variable does not exist.
The folder is in a path similar to
/home/username/.wine/c_drive/users/username/Application Data/Sketchup
You may need to set your folder viewer to 'display hidden files' in order to see the .wine folder
